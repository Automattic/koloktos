# wpboost

wpboost PECL PHP extension.

Contains various WordPress utility functions.

The most basic idea with adding these functions is to improve performance.

The filename `wpboost.php` contains polyfills in PHP.

## Building

To build:
1. Run `phpize`
2. Run `./configure`
3. Run `make`
4. Run `make test`
